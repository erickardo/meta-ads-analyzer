# Meta Ads Analyzer

Meta Ads Analyzer is a beautifully designed, modern web application built for entrepreneurs and marketers to analyze their Meta (Facebook & Instagram) ads data with clarity and precision. 

Embracing a "Modern Minimalism with Emphasis on Data" design philosophy, it strips away clutter so the most important element—your ad performance—takes center stage. 

## Features

- **Modern Minimalist UI**: Built using Radix UI and Tailwind CSS for a professional, distraction-free experience.
- **Data-Driven Dashboard**: Clear visual hierarchy focusing on ad metrics, statuses, and performance over time.
- **Fast and Responsive**: Powered by Vite and React for a lightning-fast frontend.
- **Full-Stack Capable**: Includes an Express server setup alongside the frontend for potential backend integrations.

## Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Radix UI Primitives, Framer Motion
- **Backend**: Node.js, Express
- **Build & Tools**: TypeScript, ESBuild, Prettier

## Getting Started

### Prerequisites

Identify that you have [Node.js](https://nodejs.org/) (v20+ recommended) and [pnpm](https://pnpm.io/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd meta-ads-analyzer
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

### Running the App

To start the development server:

```bash
pnpm run dev
```

The app will be available at `http://localhost:5000` (or the port specified by Vite).

### Building for Production

To build the client and server for production:

```bash
pnpm run build
```

To start the production server:

```bash
pnpm run start
```
